# Laboratorio-Hackers-Labs-DRAGON-
Este repositorio documenta un ejercicio completo de Penetration Testing (Pentesting) realizado sobre la máquina virtual "La Máquina del Dragón" (The Hackers Labs - ID 124). El informe detalla la metodología seguida para obtener acceso de bajo privilegio y, posteriormente, escalar a privilegios de administrador (root).
